# KPI & Pricing

## 1. North Star Metric

**Approved Design Value**  
고객이 승인한 디자인안의 견적 총액.

이 지표는 단순 사용량보다 실제 비즈니스 성과를 잘 반영한다.

## 2. 제품 KPI

| 단계 | KPI |
|---|---|
| Activation | 첫 프로젝트 생성, 첫 공간 스캔, 첫 디자인안 생성 |
| Engagement | 프로젝트당 생성 세션 수, 버전 수, 코멘트 수 |
| Quality | 디자이너 채택률, 고객 승인률, 제약 위반 감소율 |
| Monetization | seat 매출, 크레딧 매출, 프로젝트 과금, 마켓 수수료 |
| Retention | 월간 활성 조직, 반복 프로젝트 수, 갱신율 |

## 3. 파일럿 가격안

### Design Studio Pilot
- 기간: 8주.
- 비용: ₩3,000,000~₩8,000,000.
- 포함: 계정 3~5개, 프로젝트 10개, 온보딩, 커스텀 템플릿 1개.

### Brand Catalog Pilot
- 기간: 12주.
- 비용: ₩10,000,000~₩30,000,000.
- 포함: 제품 50~200개 3D 카탈로그화, XR 배치, 리드/견적 연동.

### Exhibition Pilot
- 기간: 프로젝트 단위.
- 비용: ₩5,000,000~₩20,000,000.
- 포함: 전시 도면 업로드, 부스 배치, 렌더, 견적/BOM.

## 4. 가격 방어 논리

- 렌더 외주 비용 절감.
- 수정 라운드 감소.
- 고객 미팅에서 즉시 설득 가능.
- 제안서/계약 전환율 상승.
- 제품 판매/업셀 증가.
- 치수 오류와 반품/재시공 리스크 감소.

## 5. 크레딧 과금 예시

| 작업 | 크레딧 |
|---|---:|
| 간단한 3D 에셋 생성 | 5~20 |
| 고품질 3D 에셋 생성 | 30~100 |
| 고해상도 텍스처 | 10~30 |
| 전체 레이아웃 자동 생성 | 20~80 |
| 고품질 렌더 1장 | 5~25 |
| CAD/BIM export | 20~200 |
