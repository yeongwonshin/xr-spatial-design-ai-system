# Implementation Roadmap

## Phase 0: Research & Design Validation — 4~6주

목표:
- 핵심 고객군 20~30명 인터뷰.
- 카페/쇼룸/전시 3개 도메인 워크플로우 분석.
- XR 스케치+음성 입력 프로토타입 검증.

산출물:
- 고객 문제 정의.
- 디자인 룰셋 초안.
- 기술 PoC.
- 가격 가설.

## Phase 1: Professional Prototype — 8~12주

기능:
- Vision Pro 또는 iPad 기반 공간 스캔.
- 3D 스케치 입력.
- 음성/텍스트 프롬프트.
- 제약 JSON 컴파일.
- 외부 3D 생성 API 연동.
- 기본 충돌/치수 검증.
- GLB/USDZ 출력.

성공 기준:
- 현장 미팅에서 10분 이내 첫 디자인안 생성.
- 5개 파일럿 고객의 반복 사용.

## Phase 2: B2B Studio Beta — 3~4개월

기능:
- 프로젝트/팀/권한.
- 버전 관리와 코멘트.
- 브랜드/일반 에셋 라이브러리.
- 견적/BOM 초안.
- 카페/전시/리테일 템플릿.
- Web review viewer.

성공 기준:
- 유료 파일럿 10곳.
- 프로젝트당 평균 3개 이상 생성/수정 세션.
- 제안서 전환율 개선 사례 확보.

## Phase 3: Commercial Launch — 6~9개월

기능:
- 구독/크레딧 결제.
- 마켓플레이스 베타.
- CAD/BIM export.
- 고급 검증 룰셋.
- CRM/견적 시스템 연동.
- 보안/감사/SSO.

성공 기준:
- MRR 확보.
- 브랜드 카탈로그 파트너 5곳 이상.
- 생성 비용 대비 gross margin 안정화.

## Phase 4: Enterprise Platform — 12개월+

기능:
- 전용 모델/전용 카탈로그.
- 프랜차이즈 매장 템플릿.
- 대형 전시/리테일 프로젝트 지원.
- 온프레미스/프라이빗 클라우드 옵션.
- 고도화된 BIM/ERP 연동.

성공 기준:
- Enterprise 계약.
- 반복 프로젝트 템플릿화.
- 마켓플레이스 거래 매출 확대.
