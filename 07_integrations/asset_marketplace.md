# Asset Marketplace

## 1. 마켓플레이스 역할

RoomForge XR의 마켓플레이스는 단순 3D 파일 판매소가 아니다. 실제 공간 디자인 생성과 견적에 연결되는 **구매 가능 3D 제품 그래프**다.

## 2. 에셋 유형

| 유형 | 설명 |
|---|---|
| Generic assets | 일반 가구/소품/조명 |
| Brand catalog assets | 실제 판매 제품 SKU와 연결 |
| Designer templates | 디자이너가 만든 공간 템플릿 |
| Exhibition modules | 부스/파티션/조명/사인 모듈 |
| Generated assets | 사용자가 생성한 커스텀 모델 |
| Material libraries | 재질, 마감, 텍스처 |

## 3. 에셋 메타데이터

필수:
- 카테고리.
- 실제 치수.
- 스타일 태그.
- 소재/색상.
- 가격/통화.
- 납기.
- 재고 상태.
- 라이선스.
- 지원 포맷.
- LOD 정보.
- 설치 조건.

## 4. 검색/추천

랭킹 요소:
- 공간 치수 적합성.
- 사용자 스타일 프롬프트 일치.
- 예산 적합성.
- 브랜드 우선순위.
- 재고/납기.
- 과거 승인 프로젝트 유사도.
- 마진/수수료 최적화.

## 5. 브랜드 입점 모델

- 3D 카탈로그 변환비.
- 월 입점비.
- 판매/리드 수수료.
- 추천 노출 광고.
- 프랜차이즈 패키지 템플릿.

## 6. 품질 기준

- 실제 제품 치수와 3D 모델 치수 일치.
- PBR 텍스처 포함.
- XR 프리뷰용 저용량 LOD 포함.
- CAD/BIM 메타데이터 포함.
- 상업 사용 권리 확인.
- SKU/가격/재고 API 업데이트.
