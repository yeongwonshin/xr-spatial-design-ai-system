# Personas & Use Cases

## 1. 핵심 페르소나

### A. 인테리어 실장 / 디자인 PM
- 목표: 고객 미팅에서 빠르게 설득력 있는 공간안을 제시.
- 고통: 고객이 말로 설명한 취향을 정확히 파악하기 어렵고, 렌더 외주/수정 시간이 길다.
- 기대 기능: 현장 스캔, 즉석 배치, 스타일 변환, 견적 자동화, 고객 승인 링크.

### B. 건축사무소 주니어/시니어 디자이너
- 목표: 초기 콘셉트 탐색과 클라이언트 커뮤니케이션 효율화.
- 고통: 2D 도면과 말로는 스케일 감각을 전달하기 어렵다.
- 기대 기능: 매스/가구/동선 스케치, BIM 연동, 버전 비교, 스케일 검증.

### C. 가구 브랜드 B2B 영업팀
- 목표: 고객 공간에 맞는 제품 조합을 추천하고 구매 전환.
- 고통: 카탈로그 이미지는 실제 공간 적합성을 보여주기 어렵다.
- 기대 기능: 브랜드 에셋 DB, 실제 치수 배치, 번들 견적, 재고/납기 연동.

### D. 전시·팝업 기획자
- 목표: 부스, 동선, 조명, 포토존을 빠르게 배치하고 승인받기.
- 고통: 현장 제약과 브랜드 요구사항을 동시에 만족시키는 대안 생성이 어렵다.
- 기대 기능: 관람객 동선 시뮬레이션, 시야각 확인, 부스 모듈 라이브러리, 설치 견적.

### E. 게임/콘텐츠 제작자
- 목표: XR/게임 공간의 레벨 블로킹과 3D 에셋 제작 속도 향상.
- 고통: 아이디어에서 플레이 가능한 3D 환경까지 시간이 오래 걸린다.
- 기대 기능: 스케치 기반 레벨 블록아웃, 스타일 에셋 생성, 최적화된 GLB/FBX 출력.

## 2. 전문 유스케이스

### Use Case 1: 카페 인테리어 현장 제안
1. 고객 매장 방문.
2. Vision Pro 또는 모바일 LiDAR로 공간 스캔.
3. “입구 쪽은 대기 공간, 창가에는 2인석, 안쪽은 바 테이블”이라고 말하며 대략 선으로 동선을 그림.
4. 시스템이 카페 테이블, 의자, 조명, 카운터를 자동 생성/배치.
5. 통로 폭, 좌석 수, 콘센트 위치, 예산을 검증.
6. 고객에게 XR 미리보기와 견적서를 공유.

### Use Case 2: 브랜드 쇼룸 가구 판매
1. 고객 집 거실을 스캔.
2. 사용자가 “북유럽풍 3인 소파와 낮은 원목 테이블” 요청.
3. AI가 브랜드 카탈로그에서 실제 판매 제품을 우선 추천.
4. 공간에 배치 후 치수/동선/색상 조합 검증.
5. 구매 링크 또는 영업 견적서 발행.

### Use Case 3: 전시 부스 설계
1. 전시장 도면 또는 현장 스캔 업로드.
2. 부스 구획, LED 월, 상담 테이블, 체험존을 펜으로 지정.
3. “미래형, 흰색, 곡선형, 20명 체류 가능” 프롬프트 입력.
4. AI가 여러 부스안을 생성.
5. 소방법/피난 동선/시야각/설치 시간/예산 점검.
6. 클라이언트 승인 후 CAD, 렌더, 자재 리스트 출력.

## 3. 구매 의사결정자

| 고객군 | 사용자 | 구매자 | 예산 논리 |
|---|---|---|---|
| 인테리어 업체 | 디자이너, 실장 | 대표, 영업총괄 | 계약 전환율, 수정 비용 절감 |
| 건축사무소 | 설계팀 | 파트너, BIM 리드 | 초기 설계 효율, 고객 커뮤니케이션 |
| 가구 브랜드 | 매장/영업팀 | 리테일/커머스 임원 | 전환율, 객단가, 반품 감소 |
| 전시기획사 | 기획/디자인팀 | 프로젝트 디렉터 | 제안서 속도, 승인율 |
| 콘텐츠 스튜디오 | 레벨 디자이너 | 제작총괄 | 에셋 제작 비용, 반복 속도 |
