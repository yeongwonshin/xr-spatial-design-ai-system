# Business Model

## 1. 수익 모델

### A. B2B SaaS 구독

| 플랜 | 대상 | 월 과금 예시 | 포함 기능 |
|---|---|---:|---|
| Starter Studio | 소형 인테리어 업체 | ₩199,000/seat | 공간 스캔, 월 생성 크레딧, 기본 견적 |
| Pro Studio | 중대형 디자인사 | ₩599,000/seat | 협업, 브랜드 카탈로그, CAD/BIM 출력 |
| Enterprise | 가구 브랜드/프랜차이즈/대형 설계사 | 협의 | SSO, 전용 모델, API, 보안, 온프레미스 옵션 |

### B. 프로젝트당 과금
- 카페/상업공간 제안 패키지: 프로젝트당 ₩150,000~₩700,000.
- 전시 부스/쇼룸 프로젝트: 프로젝트당 ₩500,000~₩3,000,000.
- 고해상도 렌더/도면/BIM 변환은 추가 과금.

### C. 생성 크레딧
- 3D 모델 생성, 고품질 리토폴로지, PBR 텍스처, 물리 검증, 렌더링에 크레딧 사용.
- 외부 모델 제공자 API 비용을 마진 포함 재판매.

### D. 3D 에셋 마켓플레이스
- 가구 브랜드 입점 수수료.
- 유료 에셋 판매 수수료 20~35%.
- 커스텀 브랜드 카탈로그 구축비.

### E. 커머스/견적 연동 수수료
- 실제 가구 구매, 자재 발주, 시공 견적 연결 시 거래 수수료.
- B2B 리드 판매 또는 파트너 수수료.

## 2. 매출 확대 전략

### Land
- 인테리어 업체의 고객 미팅용 XR 제안 도구로 진입.
- “계약 전환율 향상”을 ROI 메시지로 사용.

### Expand
- 견적서, 시공관리, 가구 발주, 자재 카탈로그로 확장.
- 브랜드/프랜차이즈 템플릿을 제공해 반복 프로젝트를 흡수.

### Lock-in
- 프로젝트 히스토리, 고객별 취향 데이터, 브랜드 에셋 DB, 승인 로그를 축적.
- CAD/BIM/견적 시스템과 깊게 연동해 전환 비용을 높임.

## 3. 가격 책정의 기준 KPI

- 고객 미팅 1회당 제안안 생성 시간 절감.
- 계약 전환율 상승.
- 수정 라운드 감소.
- 외주 렌더링 비용 절감.
- 평균 객단가/추가 판매 증가.
- 반품/치수 오류/시공 변경 비용 감소.

## 4. 시장 진입 우선순위

1순위: **상업 인테리어 업체**  
이유: 고객 설득, 견적, 시공 가능성 검증이 즉시 돈으로 연결된다.

2순위: **전시/팝업 기획사**  
이유: 단기 프로젝트가 많고, 빠른 제안과 승인 워크플로우가 중요하다.

3순위: **가구 브랜드/리테일**  
이유: 커머스 전환과 브랜드 에셋 마켓플레이스 확장성이 높다.

4순위: **건축사무소**  
이유: 도입 장벽은 높지만 BIM/CAD 연동 이후 장기 락인이 강하다.
