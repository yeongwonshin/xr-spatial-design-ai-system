# Competitive Positioning

## 1. 경쟁군

| 경쟁군 | 대표 형태 | 한계 | RoomForge XR의 차별점 |
|---|---|---|---|
| AI 인테리어 렌더링 | 사진 업로드 후 스타일 변환 | 실제 3D 치수/동선/구매와 분리 | 스캔 기반 3D 배치와 견적 연동 |
| 3D 모델 생성 AI | 텍스트/이미지로 에셋 생성 | 공간 제약과 프로젝트 워크플로우 부족 | 공간 스케치 제약 + 배치 검증 |
| CAD/BIM 도구 | 전문 설계와 문서화 | 초기 고객 협업이 어렵고 진입장벽 높음 | XR 스케치로 시작해 CAD/BIM으로 출력 |
| AR 가구 배치 앱 | 단일 제품 배치 | 전체 공간 디자인/견적/협업 부족 | 프로젝트 단위 디자인 시스템 |
| XR 협업 도구 | 3D 리뷰/회의 | 생성형 설계 기능 부족 | 생성·검증·견적까지 폐쇄 루프 |

## 2. 핵심 포지션

**“사진을 예쁘게 바꾸는 AI”가 아니라, “실제 공간에 설치 가능한 3D 디자인안을 생성하고 판매/시공까지 연결하는 XR 운영체제”로 포지셔닝한다.**

## 3. 차별화 축

1. 공간 제약 기반 생성.
2. B2B 워크플로우 중심.
3. 실제 제품 카탈로그와 연결.
4. 견적·발주·CAD/BIM 내보내기.
5. XR 협업과 승인 기록.

## 4. 방어력

- 누적 공간-디자인-견적 데이터.
- 브랜드 카탈로그/가격/재고 연동 네트워크.
- 전문 도메인별 룰셋: 카페, 리테일, 전시, 오피스, 주거.
- 사용자 프로젝트 히스토리와 팀 협업 데이터.
- CAD/BIM/ERP/커머스 연동 생태계.
