# Safety & Quality Guardrails

## 1. 제품 안전 원칙

RoomForge XR는 전문 디자이너와 시공사의 판단을 보조하는 시스템이다. 자동 생성 결과를 “시공 확정 도면”으로 오인하지 않도록 명확한 검증 상태를 표시한다.

## 2. Guardrail 유형

### A. 물리적 안전
- 문, 비상구, 소화전, 배전반을 가리지 않음.
- 최소 통로 폭 미달 경고.
- 무거운 설치물의 벽체 고정 가능성 표시.
- 천장 조명/행잉 오브젝트의 높이와 하중 경고.

### B. 치수/설치 품질
- 실제 판매 제품과 생성 모델 치수 차이 표시.
- 비표준 치수는 “커스텀 제작 필요”로 태깅.
- 문 개폐 반경, 의자 후퇴 공간, 카운터 작업 공간 검증.

### C. 상업적 정확성
- 가격/재고/납기가 확인되지 않은 제품은 “예상 견적”으로 표시.
- 브랜드 카탈로그의 최신 가격과 불일치 시 경고.
- 생성형 에셋은 실제 구매 가능 제품이 아님을 명시.

### D. IP/라이선스
- 업로드 에셋의 라이선스 상태 저장.
- 생성 모델 결과의 상업 사용 가능 여부 태깅.
- 브랜드 제품 형태를 무단 복제하는 프롬프트 제한.

### E. 데이터 프라이버시
- 고객 주거 공간 스캔 데이터 암호화.
- 얼굴/개인정보/문서 자동 블러 옵션.
- 외부 생성 API로 전송되는 데이터 최소화.

## 3. 품질 상태 라벨

| 라벨 | 의미 |
|---|---|
| Concept | 아이디어 수준, 치수/시공 검증 미완료 |
| Spatially Checked | 충돌/동선/치수 검증 완료 |
| Quote Ready | 제품/수량/단가 연결 완료 |
| Procurement Ready | 구매/발주 가능 |
| Construction Review Needed | 전문가 검토 필요 |

## 4. 전문 사용자 권한

- 디자이너: 생성, 수정, 승인 요청.
- PM: 견적 확정, 고객 공유.
- 시공사: 설치성 코멘트, 위험 표시.
- 고객: 보기, 코멘트, 승인.
- 관리자: 카탈로그, 가격, 권한, 보안 관리.
